# HULA vs ECMP: P4 Load Balancing Study

Dynamic composability and performance analysis of latency-aware load balancing in P4 data planes.

## Overview
This project implements and compares two load balancing schemes:
- **ECMP**: Hash-based equal-cost multipath routing
- **HULA**: Hop-by-hop Utilization-based Load Adaptive routing

## Setup

### Prerequisites
